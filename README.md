# yongling.info

Personal website of Yongling Zhao — Senior Scientist, ETH Zürich.

- Single self-contained static site: everything lives in `index.html` (inline CSS + JS).
- Hosted on GitHub Pages with the custom domain `yongling.info` (see `CNAME`).
- To update: edit `index.html`, commit, push — the live site refreshes automatically in ~1 minute.
